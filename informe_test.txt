DESAFÍOS ENCONTRADOS Y SOLUCIONES
---------------------------------

1. COMPATIBILIDAD DE VERSIONES:
   - Problema: Node.js v12 incompatible con Prisma
   - Solución: Actualización a Node.js v18.20.8 usando nvm

2. CONFIGURACIÓN TYPESCRIPT:
   - Problema: Target ES5 causaba errores de sintaxis
   - Solución: Cambio a target ES2020 en tsconfig.json

3. TESTING DE REACT:
   - Problema: Incompatibilidad de jsdom con Jest
   - Solución: Instalación de versiones compatibles

4. SELECTORES DE TESTS:
   - Problema: Tests buscaban texto en inglés en UI en español
   - Solución: Ajuste de todos los selectores para usar texto español

5. VISUALIZACIÓN DE REPORTES:
   - Problema: Dashboard HTML no se cargaba correctamente
   - Solución: Implementación de servidor web y simplificación de CSS
